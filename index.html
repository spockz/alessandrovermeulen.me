Hello

My name is Alessandro and I've been a functional programming enthousiast for over 8 years. First in Haskell now in Scala. I'm a Functional Programming evangelist at ING Bank where I maintain core libraries and frameworks for the API Platform and train colleagues in functional and reactive programming with Scala and Akka.

My interests are in all things related to programming language design, compilers, and resilient systems. I get excited by opportunities to remove redundant code through abstraction.

This blog exists so I can write down interesting tidbits I stumble upon and to share the ideas I occasionally have.
